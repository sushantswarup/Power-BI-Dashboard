Real-Time Twitter Analysis

Introduction

  • Purpose

    This report provides a comprehensive overview of the data analysis and visualization tasks completed during my internship at NullClass Pvt Ltd. The report aims to document the objectives, methodologies, and outcomes of various data visualization projects, highlighting how these tasks enhanced our understanding of Twitter data and improved reporting capabilities.

  • Scope

    The scope of this report includes the creation of multiple visualizations using Power BI. These visualizations encompass line charts to track engagement trends, comparative analyses of engagement rates, dual-axis charts to illustrate media interactions, and pie charts to represent click proportions. Each task was designed to address specific aspects of social media data analysis and to demonstrate proficiency in using Power BI for data-driven insights.

Background

  • Context

    This project was part of a broader initiative focused on analyzing Twitter data to extract valuable insights. The internship was conducted at NullClass Pvt Ltd, a company recognized for its expertise in data analytics and visualization. This setting provided a platform to apply theoretical knowledge to practical scenarios, utilizing real-world data to solve complex problems.

  • Organization Overview

    NullClass Pvt Ltd is a leading organization in the field of data analytics, specializing in offering data-driven solutions across various industries. The company leverages advanced tools and methodologies to deliver actionable insights. During my internship, I worked with diverse datasets, gaining hands-on experience in data analysis and visualization.

Learning Objectives

  • Data Visualization

    A primary objective was to develop a deep understanding of data visualization techniques using Power BI. This included learning how to create and customize various charts and graphs to effectively communicate data insights.

  • Data Analysis

    The internship focused on enhancing my ability to analyze social media data, particularly engagement metrics and interaction patterns. I learned to interpret data trends, compare different engagement metrics, and draw meaningful conclusions from the analysis.

  • Tool Utilization

    Another key objective was gaining practical experience in using Power BI, a leading data visualization tool. This involved mastering its features, such as filtering, aggregating data, and creating interactive dashboards that facilitate data exploration and decision-making.

Activities and Tasks

  • Task 1: Line Chart and Visualization

    Created a line chart to illustrate the trend of average engagement rates for tweets over each month, distinguishing between tweets with media content and those without.

  • Task 2: Engagement Rate Comparison

    Developed a visualization comparing the number of replies, retweets, and likes for tweets with media engagements exceeding the median value. Included a filter to focus on tweets from the last six months.

  • Task 3: Dual-Axis Chart

    Analyzed tweets to compare engagement rates between tweets with app opens and those without, focusing on tweets posted between 9 AM and 5 PM on weekdays.

  • Task 4: Pie Chart with Drill-Down

    Created a dual-axis chart showing the number of media views and media engagements by day of the week for the last quarter, highlighting days with significant spikes in interactions.

  • Task 5: Click Proportions

    Designed a pie chart representing the proportion of total clicks (URL clicks, user profile clicks, hashtag clicks) for tweets with more than 500 impressions. Included a drill-down feature to view specific click types for each tweet.
